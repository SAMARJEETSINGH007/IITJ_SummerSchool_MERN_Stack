Selectors
.card, .avatar, .title, .socials, button – Class selectors used to target specific elements with those class names.
body – Element selector targeting the <body> tag.
.socials a – Descendant selector targets all <a> inside an element with class socials.
.socials a:hover, button:hover – Pseudo-class selectors for styling elements when hovered.


CSS Attributes

Box Model
padding – Adds space inside the element, between content and border.
margin – Adds space outside the element, between it and other elements.
width, height – Defines the size of elements.
border-radius – Rounds the corners of elements (like avatars or buttons).

Visual Styling
background, background-color – Sets the background color of elements.
box-shadow – Adds shadow to create depth or elevation.
color – Sets text color.
font-size, font-family – Controls the appearance of text.
text-align – Aligns text inside the element.

Layout & Alignment
display: flex – Enables flexbox layout to align elements.

justify-content, align-items – Align items horizontally or vertically in flexbox.

object-fit: cover – Ensures images fill the container without distortion.

Interactivity
cursor: pointer – Changes cursor to pointer on hover.
transition – Smoothly animates property changes like color or background.
:hover – Applies styles when user hovers over an element (like a button or icon).
